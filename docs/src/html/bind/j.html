属性J
Jet中使用属性J将Jet中的数据和UI绑定。无论是字符串，数字，json还是数组，无论是普通文本还是输入框，都是使用属性J进行绑定。 Jet会根据数据的类型和html元素的类型自动选择对应的绑定类型。 (绑定类型见下一节)
以下是一个属性绑定的例子
<div j='user'> <div j='name'> <span j='first'></span>-<span j='last'></span> </div> <div j='age'></div> </div> <div j='list'> <div j='$value'></div> </div> <script> new Jet({ data:{//数据 user:{ name:{ first:'first', last:'last' }, age:20 }, list:['l1','l2','l3'] } }); </script>
