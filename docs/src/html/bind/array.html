1.原生数组方法
Jet对原生的数组方法进行了重写，使其可以支持Jet的双向数据绑定。在Jet中也可以使用 = 操作符直接对数组（或者是json格式的数据）进行赋值。双向绑定都会被触发。Jet重写了以下八个数组的原生操作方法，并没有改变其原生的用法：
1.= (直接赋值)
2.push
3.pop
4.splice
5.shift
6.unshift
7.reverse
8.sort
2.Jet数组方法
循环类型的for元素的绑定值是一个数组。除了原生的数组操作方法，Jet 对for元素的数组还定义了一些新的操作方法，来代替或弥补原生js的数组操作方法。一共有以下九种： (点及下面的列表可以跳转到对应位置)
1.$push (向数组尾部插入一个元素)
2.$pushArray (向数组尾部插入多个元素)
3.$prep (向数组头部插入一个元素)
4.$prepArray (向数组尾部插入多个元素)
5.$insert (向数组指定位置插入一个元素)
6.$insertArray (向数组指定位置插入多个元素)
7.$remove (删除数组中的一个或多个元素)
8.$removeByIndex (删除数组中指定位置的一个或多个元素)
9.$clear (清空数组)
10.$replace (替换数组，可用于数组整体赋值)
1.$push
使用 .$push() 向数组尾部插入一个元素。请看一个例子：
<input type="button" jon='click:push' value='添加元素'> <div j='list_push' jfor-inline> <span j='$value'></span> </div> <script> new Jet({ data:{ list_push:[11,22] }, func:{ push:function(){ this.list_push.$push(this.list_push.last()+11); } } }); </script>
