get：获取模块
使用 get 方法可以获取一个已经经过 import 加载过的已经存在在 module中的模块。 在Jet元素中可以使用this.$get。在使用get 之前要保证模块已经被加载过了，否则回返回undefined。
get可以包含一个或多个参数，当使用一个参数时，直接返回一个模块。但参数有多个时，会返回键值对形式的模块集合。 用法如下：
//使用Jet.get() var module1=Jet.get('m1');//一个参数，直接返回对应的模块 var module1=Jet.get('m1','m2');//多个参数，返回键值对形式的模块集合。{m1:模块一,m2:模块2} var module1=Jet.get('m1 as my1','m2 as my1');//多个参数使用as，返回键值对形式的模块集合。{my1:模块一,my2:模块2} //使用Jet.prototype.$get() new Jet({ data:{ module1:null, }, onmounted:{ this.module1=this.$get('m1');//参数用法与Jet.get()一致 } })
