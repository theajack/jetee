prototype扩展
Jet对dom元素，数组和字符串的prototype进行了扩展，添加了一些常用的方法供开发者使用。不过为了尽量不引起命名冲突，所有的扩展方法都带有 _JT_ 的前缀。 这里只做介绍，详细的使用文档请参考Jetterjs开发文档。
dom元素的扩展：
_JT_css (获取或修改样式)
_JT_attr (获取或修改属性值)
_JT_hasAttr (判断是否含有某属性)
_JT_removeAttr (移除某属性)
_JT_findTag (根据标签查找所有孩子元素)
_JT_findAttr (根据属性查找所有孩子元素)
_JT_select (根据选择器查找所有孩子元素)
_JT_findClass (根据类名查找所有孩子元素)
_JT_addClass (添加类)
_JT_removeClass (删除类)
_JT_hasClass (判断是否含有某类)
_JT_val (获取或修改value值)
_JT_txt (获取或修改innerText值)
_JT_content (获取或修改value或者innerText值)
_JT_html (获取或修改innerHTML值)
_JT_allHtml (获取或修改outerHTML值)
_JT_next (返回下一个元素)
_JT_prev (返回上一个元素)
_JT_child (返回某个或所有孩子元素)
_JT_clone (克隆元素)
_JT_parent (返回父元素)
_JT_prepend (在第一个位置插入一个或多个元素)
_JT_append (在最后一个位置插入一个或多个元素)
_JT_toArray (将元素或者元素集合转化为一个数组)
_JT_index (获取元素在父元素中的位置)
_JT_event (为元素绑定事件)
_JT_on (为元素绑定事件)
_JT_clk (为元素绑定点击事件)
_JT_empty (清空元素下的所有内容)
_JT_remove (删除自身)
_JT_each (对元素集合进行遍历)
_JT_last (获取元素集合的最后一个元素)
_JT_first (获取元素集合的第一个元素)
_JT_exist (判断获取的元素是否存在)
数组的扩展：
_JT_each (遍历数组，相当于forEach)
_JT_empty (清空数组)
_JT_last (获取最后一个元素)
_JT_first (获取第一个元素)
_JT_remove (删除某个元素)
_JT_removeByIndex (删除某位置上的元素)
_JT_insert (插入元素)
_JT_insertArray (插入多个元素)
_JT_prepend (头部插入元素)
字符串的扩展：
_JT_has (是否包含子串)
_JT_replaceAll (替换所有元素)
Jetterjs 中还有很多方法这里没有提取使用，提取以上方法是因为在 Jet 中用到了这些功能，如果对 Jetterjs 完整的开发接口感兴趣，请访问Jetterjs开发文档
引入JUI时的原型扩展
为了方便JUI的开发，在JUI库中，我们提取了Jetterjs中几乎所有的dom操作方法，并且都去除了 _JT_ 前缀， 若是您引入了JUI那么您可以使用这些方法。详情请参考Jetterjs开发文档
