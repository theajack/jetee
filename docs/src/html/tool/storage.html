$storage()
Jet中封装了操作localStorage方法，在Jet方法中，可以直接使用 this.$storage() 调用。该方法接受两个个参数，当只使用一个参数时，表示从localStorage从获取该键对应的值，当时用两个参数时，表示设置localStorage值。 支持任何类型的值。
* 具体使用如下
* this.$storage()：获取所有值
* this.$storage(null)：清空所有值
* this.$storage('a',1) :存储一个值
* this.$storage('a') :获取一个值
* this.$storage('a',null) :清除一个值
* this.$storage({
* a:1,
* b:{b1:11}
* }) :以json格式存储值
*
* 可以存储任何类型的值，除了null和undefined
请看一个例子：
键：<input type='text' j='key'/> 值：<input type='text' j='value'/> <button jon='setStorage'>设置localStorage</button> <button jon='getStorage'>获取localStorage</button> <div jdom='storage'></div> <script> new Jet({ data:{ key:"", value:"" },func:{ setStorage:function(){ this.$storage(this.key,this.value); this.$dom.storage.html='设置成功'; }, getStorage:function(){ this.$dom.storage.html=this.$storage(this.key); } } }); </script>

$storage.session()
$storage方法默认使用localStorage，若是您想使用 sessionStorage，请使用 $storage.session()方法，该方法与 Jet.$.storage.session 方法等价。其他使用与 $storage 方法一致。
