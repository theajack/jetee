$cookie()
Jet中封装了操作cookie方法，在Jet方法中，可以直接使用 this.$cookie() 调用。该方法接受四个参数，当只使用一个参数时，表示从cookie从获取该键对应的值，当时用两个或两个以上参数时，表示设置一个cookie值。 其中第一个参数是键，第二个是值，第三个是过期时间，默认值为1，单位为天，第四个是路径，默认值为 / (全局)。
* 具体使用如下：
* $.cookie('a',1) :存储一个值
* $.cookie('a') :获取一个值
* $.cookie('a',null) :清除一个值
* $.cookie({
* a:1,
* b:{b1:11}
* }) :以json格式存储值
* $.cookie('a',1,1,'/'):后面两个参数可选，第一个是过期时间，单位为天。第二个是cookie起作用的目录，默认为当前目录。'/'意思是设置为根目录
请看一个例子：
键：<input type='text' j='key'/> 值：<input type='text' j='value'/> <button jon='setCookie'>设置cookie</button> <button jon='getCookie'>获取cookie</button> <div jdom='cookie'></div> <script> new Jet({ data:{ key:"", value:"" },func:{ setCookie:function(){ this.$dom.cookie.html=this.$cookie(this.key,this.value); }, getCookie:function(){ this.$dom.cookie.html=this.$cookie(this.key); } } }); </script>
