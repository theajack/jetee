Jet.lang.type
使用 Jet.lang.type 可以获取当前的语言，并且可以设置语言。请看一个例子：
index.js 或者 index.html 中插入以下代码：
Jet.lang.use(['cn','en']);
Jet组件中：
<select j='type' jon='change:changeLang'> <option value="cn">中文</option> <option value="en">英文</option> </select> <div jlang> <span jname='cn'>你好，世界</span> <span jname='en'>hello world</span> </div> <script> new Jet({ onmounted:function(){ this.type=Jet.lang.type; }, data:{ type:'', },func:{ changeLang:function(){ Jet.lang.type=this.type; } } }) </script>
以下是输出的html:
你好，世界 hello world
