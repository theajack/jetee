动态元素
使用 JL() 或者 new Jet.lang() 方法可以定义一个支持国际化语言的动态的Jet数据。请看一个例子：
index.js 或者 index.html 中插入以下代码：
Jet.lang.use(['cn','en']);
Jet组件中：
<span j='message'></span> <span j='message2'></span><br> <button jon='click:Jet.lang.type="cn"'>设置中文</button> <button jon='click:Jet.lang.type="en"'>设置英文</button> <script> new Jet({ data:{ message:JL({ en:'hello', cn:'你好' }), message2:new Jet.lang({ en:' world', cn:',世界' }) } }) </script>

使用json或数组作为数据源
除了只使用值类型外，JL还可以使用json数据或者数组来作为数据源。请看一个例子：
index.js 或者 index.html 中插入以下代码：
Jet.lang.use(['cn','en']);
Jet组件中：
<div j='user'> <div j='name'></div> <div j='height'></div> </div> <button jon='click:Jet.lang.type="cn"'>设置中文</button> <button jon='click:Jet.lang.type="en"'>设置英文</button> <script> new Jet({ data:{ user:JL({ en:{ name:'jack', height:'180cm' },cn:{ name:'杰克', height:'180厘米' } }) } }) </script>

将Jet数据整个作为数据源
也可以将整个Jet数据作为数据源。请看一个例子：
index.js 或者 index.html 中插入以下代码：
Jet.lang.use(['cn','en']);
Jet组件中：
<div j='user'> <div j='name'></div> <div j='height'></div> </div> <button jon='click:Jet.lang.type="cn"'>设置中文</button> <button jon='click:Jet.lang.type="en"'>设置英文</button> <script> new Jet({ data:JL({ en:{ user:{ name:'jack', height:'180cm' } },cn:{ user:{ name:'杰克', height:'180厘米' } } }) }) </script>
请使用在线使用来查看运行效果。值得注意的是 JL 或 Jet.lang 中不能再包含 JL 或 Jet.lang。
