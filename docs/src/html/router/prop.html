路由属性
Jet.router 对象中有许多与路由相关的属性，开发者可以使用它们来辅助开发。 这些属性都是只读的，不可以修改或者修改没有任何意义。一共有以下7个：
1.base (根目录路径)
2.trueBase (根目录路径是否是真实的文件目录，布尔类型)
3.history (是否使用history模式，布尔类型)
4.path (当前的路由路径)
5.url (当前的不包含根路径的路由路径)
6.params (链接后面的参数，json类型)
7.conf (存放资源文件的目录，json类型)
8.map (路由的映射表，json类型)
9.active (当前路由页面对应的jrouter dom元素)
10.out (路由出口对应的dom元素)
11.index (设置路由首页)
12.indexMap (路由首页对应的html地址，无需开发者设置)
