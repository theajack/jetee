Jet.router中的方法
除了前文中介绍的很重要的use和route方法，本节还将介绍Jet.router中的其他一些方法，主要有以下几个：
1.setBase() (设置路由的根目录和trueBase。trueBase参数为可选参数。该方法的作用是在使用了 Jet.res.define和Jet.router.use 功能的情况下，提前声明根目录（如果有的话），这样就能保证目录的准确性)
2.activeRouter() (激活当前页面对应jrouter，因为这个操作会在应用加载时自动由Jet调用，所以该方法适用场景是当jrouter会延迟加载时，比如在jload中。)
3.init() (初始化jrouter，使用场景是当jrouter被动态添加到页面中时。该方法接受一个dom参数，若是不传入参数，则对整个html文档中的jrouter进行初始化)
4.back() (退回上一个路由页面)
5.forward() (前进到上一个路由页面)
