路由事件
路由事件有onroute和onrouted， 这两个事件也包含在Jet生命周期函数中，开发者也可以主动调用 Jet.router.onroute() 和Jet.router.onrouted()方法 设置这两个事件，但是推荐在Jet生命周期中使用这两个事件。这两个函数都接受两个参数，第一个是一个必须的function参数，第二个是非必须的Jet类型的参数。请看一个例子
new Jet({ onmounted:function(){ //第二个参数可以不要 Jet.router.onroute(function(){ },this); Jet.router.onrouted(function(){ }); } })
