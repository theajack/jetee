Jstyle
使用Jstyle样式，可以给html元素绑定样式,使用格式是 Jstyle="style1:value1;style2:value2;..." 。请看下面一个例子：
<div jstyle='color:#f44'>单个静态样式</div> <div jstyle='color:#f44;background-color:#4f4'>多个静态样式</div> <div jstyle='color:{{$.color}}'>动态样式，使用字符串数据</div> <div jstyle='color:{{$.json.color}}'>动态样式，使用json数据</div> <div jstyle='color:"#"+{{$.json.number}}'>计算值的动态样式，使用json数据</div> <div> 改变字符串值：<input type="text" j='color'> </div> <div j='json'> <div>改变json的color值：<input type="text" j='color'></div> <div>改变json的number值：<input type="text" j='color'></div> </div> <script> new Jet({ data:{ color:'#44f', json:{ color:'#44f', number:900, } } }); </script>
