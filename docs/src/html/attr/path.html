Jpath
如前面的资源管理章节中所说， Jet中的资源分为五类，每一类都有其存放路径，如果对于资源使用相对路径（第一个字符不为'/'），那么相对的是各自对应的存放路径。
若是使用绝对路径，则是以url作为根的绝对路径。
以上两点是Jet中的资源引用路径规则。 Jpath属性用于设置避免使用这种规则的。该参数默认为true，一般情况下不需要设置，若是您不希望使用它们，请将其设置为false。
设置为false之后。首先不再支持资源管理中使用 @ 加载资源的方式。其次，对于支持使用相对路径的资源文件(主要只有 root.html)，可以使用相对路径。 对于其他使用Jet加载的文件，由于无法使用相对路径。Jet会将所有的相对路径的根指定为 /src 目录。对于绝对路径的使用方式不变。
<img src='jet.png'/> <!--指向 /src/image/jet.png--> <img src='jet.png' jpath='false'/> <!--指向 /src/jet.png-->
