Jvalid
Jet中使用 Jvalid 属性来对输入元素设置验证的规则，使用格式是 jvalid='type[]'。 放括号内的是一些特殊规则需要设置的参数，比如对数字范围或长度的限制。具体请参考验证规则说明。
您可以选择Jet中已经定义好的一些验证规则，或者自定义验证规则。 自定义验证规则的使用格式是 jvalid='express[exp]' 。请看一个例子：
<div>使用phone：<input type="text" jvalid='phone'></div> <div>使用带规定值的规则：<input type="text" jvalid='number[2]'></div> <div>使用带范围的规则：<input type="text" jvalid='range[10,20]'></div> <div>使用null来允许空值：<input type="text" jvalid='date null'></div> <div>使用自定义的规则（手机号验证）：<input type="text" jvalid='express[^([1]\d{10})$]'></div>

预定义的验证规则
Jet中预定义的验证规则有以下几种，你可以直接使用规则名来使用它们。 其中 除了null 其他都只能单独使用，null可以和其他任何规则共同使用，表示可以为空或者符合验证的规则。当与notnull共同使用时会忽略null。
1.notnull (非空值)
2.null (允许空值)
3.date (日期格式:XXXX-XX-XX)
4.email (邮箱：XXX@XX.XX)
5.idcard (身份证号：17位数字加一位数字或X)
6.decimal (小数)
7.url (链接)
9.phone (手机号码)
10.number (数字，可以使用 number[a] 来限定必须是a位数字；使用 number[a,b] 来限定数字必须是在a位到b位之间)
11.length (指定长度的字符串，后面方括号的使用规则与number一致)
12.lengthOfAny (指定长度的可包含任意字符（汉字）字符串，后面方括号的使用规则与number一致)
13.letterStart (指定长度的字母打头的字符串，后面方括号的使用规则与number一致)
14.range (指定范围的数字，后面方括号的使用规则与number一致)
15.express[exp] (自定义验证规则)
