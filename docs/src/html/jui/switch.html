切换按钮
切换按钮支持使用两个子元素，第一个子元素表示打开的情况，第二个表示关闭的情况。 每一个情况都有text和value，text用于显示，value用于设置开关。text值等于子元素的innerText，当设置了value属性是，value值等于设置的值，否则value值等于元素的text值，当元素text 当text为空且没有设置value时，第一个元素的value值为true，第二个为false。请看一个例子：
