确认框
JUI.confirm 中有以下以下几个方法：
1.success():成功类型确认框的简写
2.info():信息类型确认框的简写
3.warn():警告类型确认框的简写
4.error():错误类型确认框的简写
5.close():关闭队列首部的确认框
6.isOpen():当前是否有打开的确认框
7.clear():关闭所有确认框
确认框类型的默认值为info ,可选值有 success，info，warn，error
请看一个例子：
<button jon='allParam'>所有参数</button> <button jon='confirm'>简写</button> <button jon='close'>关闭队列首部的确认框</button> <button jon='isOpen'>查看是否有打开的确认框</button> <button jon='clear'>关闭所有确认框</button> <script> new Jet({ func:{ allParam:function(){ JUI.confirm({ title:"title", text:'text', type:'warn', html:false, //默认为false 是否将内容设置成 innerHTML onconfirm:function(){console.log('confirm')}, oncancel:function(){console.log('cancel')}, onclose:function(){console.log('close')}, }) },confirm:function(){ JUI.confirm('是否确认',function(){ JUI.msg('确认') },'info') },close:function(){ JUI.confirm.close(); },isOpen:function(){ JUI.msg(JUI.confirm.isOpen()); },clear:function(){ JUI.confirm.clear(); } } }); </script>
