颜色选择器
颜色选择器：j-color [size,disabled] 备注：alpha 为是否启用透明度
请看一个例子：
颜色1 颜色2
<div class='j-color xs' jui-bind='color' alpha></div> <div class='j-color s' jui-bind='color2'></div> <div class='j-color m' jui-bind='color2'></div> <div class='j-color l' jui-bind='color2'></div> <div class='j-color xl' jui-bind='color2'></div> <div class='j-color' disabled jui-bind='color2'></div> <span jstyle="color:{{$.color}}">颜色1</span> <span jstyle="color:{{$.color2}}">颜色2</span> <script> new Jet({ data:{ color:'rgba(231,22,32,1)', color2:'rgb(231,22,32)', } }); </script>
