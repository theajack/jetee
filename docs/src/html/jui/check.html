复选框与复选框组
请看一个例子：
<div class="j-checkbox-group" jui-bind='fruitList'> <!--使用icon--> <div class="j-checkbox" icon='pencil'>芒果</div> <div class="j-checkbox">西瓜</div> <div class="j-checkbox" disabled>橘子禁用</div> </div> <div j='fruitList' jfor-inline> <span j='$value'></span> </div> <div class="j-checkbox-group"> <div class="j-checkbox" icon='pencil'>芒果</div> <!--使用checked--> <div class="j-checkbox" checked>西瓜</div> <div class="j-checkbox" disabled>橘子禁用</div> </div> <script> new Jet({ data:{ fruitList:['芒果'] } }); </script>

checkbox group 对应的 JUI对象有以下以下几个方法：
1.selectAll():选中当前checkbox group下的所有checkbox
2.clear():清空当前checkbox group下的所有checkbox的选中状态
3.remove(arg):删除当前group中的某一个checkbox，参数可以是checkbox对象或checkbox位于checkbox group的索引
4.removeAll():删除当前group中的所有checkbox
