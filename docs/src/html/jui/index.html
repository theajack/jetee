JUI
JUI 是针对 Jet 开发的一套前端 UI 框架。它提供了一套基础样式类、栅格系统、消息弹出框和一些常用的UI组件。它能够完美的与Jet的数据绑定配合使用。 能够使开发者快速的开发web应用程序。
JUI 中的盒模型 全局设置为border-box。默认字体为微软雅黑字体。
JUI 目前还在开发之中，请关注最新版本的发布。
jui-bind
使用jui-bind属性，可以与Jet数据进行绑定。绑定的数据类型取决于JUI元素的类型。
jui-change
使用jui-change属性可以指定一个数据改变以后的回调函数。 该函数可以是一个Jet函数，全局函数或者一串js代码。该回调函数的参数是一个json数据，它有三个属性， ele：对应的dom元素；value：绑定的数据；jui：对应的jui元素。并不是所有的元素都支持jui-change属性。
Jet.prototype.$jui
对于使用了数据绑定的JUI元素，JUI会生成一个JUI对象，使用Jet.prototype.$jui()方法可以获取到这个对象， 并且可以设置这个对象的属性可以改变组件的状态、更新源数据。该方法的参数是jdom值，jdom元素或者dom元素。
jui-type
使用jui-type属性可以用于指定绑定数据的元素最终使用的数据类型，可选值有number,bool和string。目前可用的元素仅有 radio,checkbox和switch。
jui-onload
使用jui-onload属性可以在JUI组件绑定完成时触发一个事件。jui-onload 属性值可以使一段js代码或者一个Jet函数。
jui工具方法
在开发JUI的过程中，引入了Jetterjs中的大部分方法与原型扩展，于是我们将其整合到Jet.$中，只要使用了JUI，就可以通过Jet.$使用这些工具方法。 接口具体使用请访问：Jetterjs开发文档
