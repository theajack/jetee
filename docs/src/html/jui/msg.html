提示框
JUI.msg 中有以下以下几个方法：
1.success():成功提示的简写
2.info():信息提示的简写
3.warn():警告提示的简写
4.error():错误提示的简写
5.close():关闭队列首部的消息框
6.isOpen():当前是否有打开的消息框
7.clear():关闭所有消息框
消息类型的默认值为info ,可选值有 success，info，warn，error
请看一个例子：
<button jon='allParam'>所有参数</button> <button jon='success'>简写</button> <button jon='error'>独立的函数</button> <button jon='close'>关闭队列首部的消息框</button> <button jon='isOpen'>查看是否有打开的消息框</button> <button jon='clear'>关闭所有消息框</button> <script> new Jet({ func:{ allParam:function(){ JUI.msg({ text:'text', time:2000,//默认为2300 type:'warn',//默认为info ,可选值有 success，info，warn，error autoClose:true, //默认为true hover:true, //默认为true 鼠标hover时是否不会消失 html:false, //默认为false 是否将内容设置成 innerHTML call:function(){console.log('close')}, }) },success:function(){ JUI.msg('success'); },error:function(){ JUI.msg.error('error'); },close:function(){ JUI.msg.close(); },isOpen:function(){ alert(JUI.msg.isOpen()) },clear:function(){ JUI.msg.clear(); } } }); </script>
