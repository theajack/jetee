资源管理
Jet 通过 Jet.res.define() 方法统一定义。之后便可以在组件中用使用变量的方式引用资源。
Jet 中将资源分为以下五个大类:
1.html: (后缀为html的文件，为Jet中的路由页面和组件；默认路径为 /src/html)
2.js：(后缀为js的文件，为Jet中的模块和js文件；默认路径为 /src/js)
3.css：(后缀为css的文件，为Jet中的样式文件；默认路径为 /src/css)
4.image：(图片资源文件，缺省后缀名为 .jpg；默认路径为 /src/image)
5.media：(媒体资源文件，包含音频、视频和其他，缺省后缀名为 .mp3；默认路径为 /src/media)
声明资源之后，可以通过Jet.res[type][name]获取到对应的资源文件地址。其中type是资源的类别，name是自定义的资源名称。
更常用的方法是在引用资源文件时，使用 @[name] 作为引用目录，Jet会自动判断资源的类型。在引入模块和定义路由时，也可以使用这种语法。
这种资源统一声明管理的好处是，如果资源重命名或是资源需要替换或是目录层级发生改变时，只需要对 Jet.res.define() 方法进行修改就可以。
在一些复用性不是很高的资源文件上，您也可以不使用这种管理方式，比如路由页面。
请看几个例子
<script> //定义 Jet.res.define({ html:{ forLoad:'comp/for_load', //缺省后缀名为.html，可以包含子目录 page1:'page/p1.html' }, js:{ module1:'module/1.js', ajs:'a.js' }, css:{ test:'test.js' } image:{ wechat:'wechat.png', alipay:'alipay'//缺省后缀名为.jpg }, media:{ music:'audio/music.mp3', video:'video/video.mp4' } }) </script>
//使用 <style src='@test'></style><!--样式引用--> <div jload='@forLoad'></div><!--jload引用--> <img src="@wechat" alt=""><!--图片引用--> <audio src="@music"></audio><!--音频引用--> <video src="@video"></video><!--视频引用--> <script src='@ajs'></script><!--script引用--> <script> Jet.router.use({//路由引用 router:{ '/':'@page1', } }); Jet.import('@module1',function(){//模块引用 }) Jet.res.image.wechat //通过Jet.res 对象获取到文件目录 </script>
